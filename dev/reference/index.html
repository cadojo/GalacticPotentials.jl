ModelingToolkit.ODESystem — Method
ODESystem(field; var_map_to_dvs)

Construct an ODESystem from an AbstractScalarField by taking the gradient of the field's value with respect to the state variables. Symbols which describe the state derivatives can be provided via the var_map_to_dvs keyword argument. Alternatively, default state derivative symbols are used: :ẋ, :ẏ, and :ż for :x, :y, :z states, and :Δstate otherwise.

Example

ODESystem(field; var_map_to_dvs = Dict(:x => :ẋ, :y => :ẏ, :z => :ż))
